# Sales Analytics using SQL

This project contains multiple SQL queries to analyze sales data and generate useful business insights.

## Queries Included:
1. Total Revenue
2. Top 5 Products
3. Sales by Region
4. Monthly Sales Trend
5. Product Category Split
